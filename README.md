## 📖 О проекте

**Case-08: Библиотечная система** — это учебный проект по автоматизации процессов библиотеки.

Система позволяет:

- ✅ Вести учет книжного фонда
- ✅ Регистрировать читателей
- ✅ Оформлять выдачу и возврат книг
- ✅ Формировать отчеты по задолжникам
- ✅ Загружать данные из CSV-файлов

**Цель проекта:**  
Продемонстрировать полный цикл разработки: от анализа и проектирования до реализации и тестирования.

## 🚀 Запуск и проверка

### 1. Клонирование репозитория

bash
git clone https://gitverse.ru/topit/16-20_team3_Rozhmonov
cd case-08-library
`

### 2. Загрузка конфигурации 1С

1. Открыть 1С:Предприятие в режиме Конфигуратор

2. Выбрать меню Конфигурация → Загрузить конфигурацию из файла

3. Указать путь: src-1c/Library.cf

4. Применить изменения и запустить в режиме предприятия

### 3. Проверка данных

Тестовые наборы данных расположены в data/C08/:

· T1.csv — корректные данные (основной объем)

· T2.csv — второй набор для перепроверки

· golden.csv — эталонный результат

· invalid.csv — данные с ошибками

### 4. Документация

Актуальная версия документации находится в docs/

🧪 Тестирование

Приемочный чек-лист доступен в qa/acceptance_checklist.md

Основные сценарии проверки:

· Загрузка книг из CSV

· Добавление нового читателя

· Выдача книги

· Возврат книги

· Формирование отчета по задолжникам

· Обработка некорректных данных

Результаты тестирования заносятся в qa/test_report.md
